JSONtext.lvlib:Find Item.vi

Find the JSON item at the JSON Path specified.

Paths specify an item in a tree of nested Objects, and start with $.

Example JSON:

{"A":{"A1":123,"A2":4},"B":"Hi"}

At path $.A.A1 is 123

At path $.B (or just B) is "Hi"

At path $.A (or just A) is {"A1":123,"A2":4}

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

$Path or Item Name

JSON Path notation starts with a $. If not starting with a $, this input is treated as an item name (so <name> and $.<name> are equivalent).

Offset

Optional Offset in the input JSON string. Default is zero, the start of the string. This Offset should point to teh start of teh JSON of interest (or to whitespace in front of it).

error in (no error)
