JSONtext.lvlib:To JSON Text.vim

Converts a LabVIEW datatype into a JSON value.

Strings or Arrays of strings marked with a leading "<JSON>" tag in their name (including inside clusters) are treated as already being valid JSON. Strings not marked as <JSON> are quoted and escaped as a JSON String. Note that JSONtext.lvlib functions that produce JSON have their outputs labelled "<JSON>", so that they may be used as inputs to other functions.

Two types of LabVIEW Clusters are supported: either with all named elements, which are converted to JSON Objects, or all unnamed elements (or elements just named "<JSON>"), which are converted to a JSON Array. Handling of mixed clusters with some named and some unnamed in undefined (currently, this VI will use the first element to determine whether to convert to an Object or Array).

 Anything

Anything contains the data you want to convert to a UTF-8 JSON string. This input accepts most LabVIEW types, including arrays and clusters of those types.

enable LabVIEW extensions? (T)

Enables LabVIEW JSON extensions to support NaN and Inf values of floating-point numbers. Not all JSON parsers support these extensions.

Pretty Print?

Formats the ouput JSON using new lines and indentation to make it more easily human readable.
