JSONtext.lvlib:Next JSON Value.vi

Gets the next JSON Value (object, array, string, number, true, false, or null)

"Offset" should point to the start of the JSON Value, or to whitespace in front of the Value.

 JSON

Input JSON. Optional "Offset" indicates an offset in this string pointing to the start of the JSON item to operate on.

Offset

Optional Offset in the input JSON string. Default is zero, the start of the string. This Offset should point to teh start of teh JSON of interest (or to whitespace in front of it).

JSON Value Type

Rest of input

<JSON>Value

Offset past Value

Offset, in the input JSON, just past the item of interest.
